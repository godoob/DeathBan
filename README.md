# DeathBan
A HCF plugin to ban players for certain time if they die.

## Permissions
`deathban.noban`
  - With this permission players won't get banned even if they die
  - defaults to no one
  
`deathban.main` 
  - The permission for the main command of deathban (WIP!)
  - defaults to op
